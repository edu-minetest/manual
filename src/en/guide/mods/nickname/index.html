    Tips

    Before using this mod, remember to confirm that the mod has been enabled.

    Nickname mod allows to add a nickname (support unicode char) and nickname color to player in the game.

    You can set your nickname outside the gameplay:

    Others can see your nickname and the name within parentheses in multi-player world:

    Tips

    Specifying (modifying) a nickname requires the nickname permission, which can be granted to oneself through the chat command: /grantme nickname. Or authorize to a specific player: /grant [player_login_name] nickname

    Tips

    How to send chat commands, please refer to "In-Game Management: Send Commands".

    # Chat Command

    After getting nickname permission, you can set nickname and color by sending chat commands.

    # Set or get nickname

    /nickname[ your_nickname[, player_name]]

    • If you have server permissions, then you can specify nickname for other players (player_name)
    • If you don't specify a nickname, your current nickname (if any) will be displayed.
    • If a nickname is specified: your_nickname, then it is to set the nickname.

    # Set or get nickname color

    /nickname_color[ color[, player_name]]

    • If you have server permissions, then you can specify nickname colors for other players (player_name)
    • If no color is specified, then your current nickname color (if any) will be displayed.
    • If the color: color is specified, then the nickname color is set.
      • For example, specify the nickname color red: /nickname_color red

    # Set or get nickname background color

    /nickname_bgcolor[ color[, player_name]]

    • If you have server permissions, then you can specify nickname colors for other players (player_name)
    • If you don't specify a background color, your current nickname background color (if any) will be displayed.
    • If the background color: color is specified, then it is to set the nickname background color.

    # Color specification

    • Specify in the form of RGBred, green, and blue three-component. R: Red component; G: Green component; B: Blue component; A: Alpha transparency value
      • #RGB Defines a color in hexadecimal format.
      • #RGBA Defines a color in hexadecimal format and alpha channel.
      • #RRGGBB Defines a color in hexadecimal format.
      • #RRGGBBAA Defines a color in hexadecimal format and alpha channel.
    • Specify by color name (see the table below for the name)
      • Support for named colors from CSS Color Module Level 4open in new window.
      • To specify a value for the alpha channel, append #A or #AA to the end of the color name (e.g. red#08).
