    # 可汗学院 (Khan Academy)open in new window

    可汗学院是一家拥有多年在线教育经验的知名机构(非营利组织)，其教育内容和课程设计都经过了专业的研究和审核。致力于为全世界所有人提供免费的一流教育.

    open in new window

    可汗学院的宗旨是提供高质量的、免费的、个性化的在线教育资源，以帮助所有人提高其知识水平和技能。可汗学院的目标是通过使用技术来打破教育中的障碍，使教育变得更加普及和公平。该平台采用自适应学习技术，根据用户的学习进度和能力，为他们提供个性化的学习体验。此外，可汗学院的视频课程和练习题是免费的，并且每个人都可以随时随地访问它们。

    • 自适应学习：可汗学院使用自适应学习技术，根据用户的学习进度和能力，为他们提供个性化的学习体验。这使得学生可以在自己的节奏下学习，并根据自己的需要和能力进行调整。
    • 丰富的课程资源：可汗学院涵盖了广泛的学科，包括数学、科学、计算机科学、历史、艺术等等。该平台的视频课程和练习题是免费的，并且每个人都可以随时随地访问它们。此外，可汗学院还提供了一些其他的教育资源，如工具和指南。
    • 社区支持：可汗学院拥有一个庞大的社区，可以帮助学生解决他们所遇到的问题。该平台的用户可以与其他学生和志愿者导师互动，分享学习经验和资源。此外，可汗学院还提供了一些教育计划，如沙龙和学生挑战赛，以鼓励学生积极参与学习。

    # PhET 科学与数学的互动式模拟open in new window

    PhET（Physics Education Technology）是一个由科罗拉多大学波尔得分校的教师和研究人员创建的教育技术项目，旨在为学生提供免费、互动、可视化的物理学、化学、数学、地球科学和生物学等各种学科模拟器。

    PhET的模拟器是基于科学研究和与教师和学生的反馈和建议结合而成的，具有高度的互动性、可视化和个性化的特点。

    学生可以通过实验和探索来学习各种学科的概念，并可以根据自己的需要和能力来调整模拟器的参数和难度级别。PhET的模拟器可以在网站 phet.colorado.eduopen in new window 上免费获取，并支持多种语言。PhET的目标是为教育提供更好的工具，帮助学生更好地理解和掌握不同学科的概念。

    # Snap! 可视化编程open in new window

    是一款JS开发的运行在浏览器里的开源多范式可视化计算机编程语言。由Scratch语言衍生，包含并扩展了Scratch的功能。主要针对12岁以上的用户。

    • open in new window
    • 其区别于Scratch的最重要的特性为：
      • 高阶函数
      • 嵌套列表，例如列表的列表
      • 角色嵌套
      • Snap!代码可转换为其他主流程序语言，如Python, JavaScript和C等.
    • 编程之美与乐趣课程【英】—— "Beauty and Joy of Computing" (BJC) curriculumopen in new window

    # 开源离线教育平台与资源 Kolibriopen in new window

    是一款开源的教育软件，旨在为全球范围内的学习者提供高质量的教育资源和个性化学习体验。它具有离线学习、个性化学习和开放式教育资源等功能，可以帮助学生在没有互联网连接的环境下进行学习，并提供适合个人需求的学习资源和学习路径，同时也可以让教师和学生自由地创建、共享和访问教育资源。

    • 解决了教学和学习材料难以找到、有用和负担得起的问题，提供了一个庞大的开放教育资源库，可以离线访问并根据个人学习需求进行个性化定制。
    • 通过提供创建和共享互动学习内容和评估的平台，以及跟踪学生进度和识别改进领域的工具，支持教育工作者实现以学生为中心的学习。
    • 通过提供自适应学习路径和个性化反馈，可根据个人学生需求和学习风格进行定制，以解决大班教学和多元化学习者需求的挑战。
    • 通过推广开放和包容的学习环境，优先考虑边缘化社群的需求和声音，挑战在教育中持续存在的压迫性结构，以促进公平和平等的学习环境。
    • 通过提供协作和共创学习资源的工具，促进了社区对教育科技的设计和融入学习空间的所有权。
    • 通过允许学习者离线访问教育资源，并提供最低技术要求的内容创建和分享工具，解决了技术基础设施和可靠的互联网访问的成本问题。到目前为止，有135个教学资源频道可供下载,其中包括前面提到的可汗学院open in new window,少儿编程入门之 Blockly Gamesopen in new window,phET 科学与数学的互动式模拟(英)open in new window.

    # Concord Consortium Enopen in new window

    Concord Consortium是一家非营利性教育研究与开发组织，专注于为教师和学生创建创新的基于技术的教育资源。他们的使命是帮助教育工作者将技术融入教学，帮助学生通过互动和吸引人的活动更有效地学习。

    Concord Consortium的教育资源涵盖了广泛的学科，包括科学、数学、工程和技术。他们的一些知名项目包括Molecular Workbenchopen in new window，这是一款用于化学和物理教育的模拟软件，以及Common Online Data Analysis Platform（CODAP），这是一种面向学生和教师的数据分析工具。

    Concord Consortium还进行研究和评估研究，以衡量其教育资源的有效性，并为开发新工具和方法提供指导。
