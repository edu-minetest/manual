    # 常见问题

    # Windows 下无法输入

    请切换到英文输入法.

    注意

    • 已知问题: 在Windows系统上输入文字,只能使用英文输入法,不能使用任何中文输入法输入,否则文字输入不了.
    • 在Windows系统中输入文字请切换到英文输入法.
    • BTW:在Android以及Linux(Ubuntu)下输入中文正常。

    # 联系我们

    # 汇报问题或建议

    如果你有任何的关于《莽兜世界:教育版》本身的问题或建议，请点击此处,并指出具体细节.

    • 国内:新建问题open in new window
    • 国外:New Issueopen in new window

    必要时，请提供一个 最小复现 Repo[1] 连接。并提供调试日志. 如果你的问题与显示有关，请同时提供相关截图。

    # 获得帮助

    仅限《莽兜世界:教育版》本身

    • 如果你有关于 Lua, CPP 代码 的学习与使用问题，或者你不知道如何写你想要的东西，你可能需要在 相关 论坛、Stack Overflow 等地方提问。

    • 如果你有 Minetest 的相关问题，请在 Minetest 论坛open in new window提问。

    请只在仓库发起 《莽兜世界:教育版》以及与 Riceball LEE 编写的 Quzi 模组的相关讨论。

    • 如果有你遇到了使用问题但不知道如何解决，我们也欢迎你在 GitHub 上 新开一个讨论open in new window。问题总是受欢迎的，无论它们是否简单。你只需要确保三点:

      1. 你已经尝试过搜索相关文档。

      2. 你在讨论中提供了详细的描述。

        如果你不知道如何配置某些内容，请描述你想要的内容、搜索的内容或你希望看到指南的哪个部分 (以便我们改进我们的文档)。

      3. 你不是在问与 《莽兜世界:教育版》 无关的问题，也不是在寻求“技术支持”。

        我们不提供 “如何自定义某些内容” (如何编写自己的代码) 的相关帮助。 除非你捐赠此项目，否则请不要为该部分寻求帮助。

        作者目前很忙，不想“成为文档提供者”，也不太有时间“教你如何编写代码”。

    1. 最小复现 Repo 的基本思想是使用最少的代码和配置来触发丢失或错误的行为。最小复现 Repo 使开发人员更容易查看错误或缺失功能的位置，并验证新代码是否满足要求。 ↩︎
