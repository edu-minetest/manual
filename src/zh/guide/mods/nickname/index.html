    提示

    使用该模组前,记得先确认已经启用该模组.

    Nickname(昵称)模组允许在游戏内给玩家添加昵称(支持中文)以及昵称的颜色.

    提示

    指定(修改)昵称需要nickname权限,该权限可以通过聊天命令赋予自己权限: /grantme nickname. 或者授权给指定玩家:/grant [player_login_name] nickname

    提示

    如何发送聊天命令,请参阅《游戏内管理:发送命令》.

    # 聊天命令

    在获得nickname权限后,就可以通过发送聊天命令来设置昵称和颜色.

    # 设置或获得昵称聊天命令

    /nickname[ your_nickname[, player_name]]

    • 如果有server权限,那么你可以为其它玩家(player_name)指定昵称
    • 如果不指定昵称,那么就会显示你当前的昵称(如果有的话).
    • 如果指定了昵称:your_nickname,那么就是设置昵称.

    # 设置或获得昵称颜色聊天命令

    /nickname_color[ color[, player_name]]

    • 如果有server权限,那么你可以为其它玩家(player_name)指定昵称颜色
    • 如果不指定颜色,那么就会显示你当前的昵称颜色(如果有的话).
    • 如果指定了颜色:color,那么就是设置昵称颜色.
      • 例如指定昵称颜色红色: /nickname_color red

    # 设置或获得昵称背景颜色聊天命令

    /nickname_bgcolor[ color[, player_name]]

    • 如果有server权限,那么你可以为其它玩家(player_name)指定昵称颜色
    • 如果不指定背景颜色,那么就会显示你当前的昵称背景颜色(如果有的话).
    • 如果指定了背景颜色:color,那么就是设置昵称背景颜色.

    # 颜色的指定

    • 以RGB红绿蓝三分量的形式指定. R: 红色分量; G: 绿色分量; B: 蓝色分量; A: Alpha透明值
      • #RGB 以十六进制格式定义颜色。
      • #RGBA 以十六进制格式和 alpha 通道定义颜色。
      • #RRGGBB 以十六进制格式定义颜色。
      • #RRGGBBAA 以十六进制格式和 alpha 通道定义颜色。
    • 以颜色名称指定(名称详见下表)
      • 支持CSS Color Module Level 4open in new window的命名颜色.
      • 要指定 alpha 通道的值，请在颜色名称末尾附加 #A 或 #AA（例如 red#08）。
