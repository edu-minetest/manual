    # 挖掘并得到掉落的方块

    在生存模式获得资源的方式就是通过挖掘并得到掉落的方块.

    树一般由树干与树叶组成。如图所示，这是一颗苹果树，附近颜色较浅的树是松树。木头都是一样的，除了颜色与纹理上的差异。

    由于你没有任何工具，你需要靠近并(用小小的十字准心)对准木头，连续击打木头，你会发现木头出现了方块裂痕，继续不停的击打木头,当裂痕布满这个方块，这个方块会被挖掉，以物品的形式出现在地上，接近会被拾取(默认启用ItemDrop模组)。大部分方块都需要特定的工具才可以快速破坏并掉落。

    # 合成物品与设备

    打开物品栏，将木头左键，放入合成下面的6个方格中的任意一个，合成好的物品就出现在右边的那个小方格中，恭喜你，你把木头分解为了4个木板。 然后你可以把右边小方格中合成好的木板,拖到自己的随身物品栏中.

    然后放块木板到合成栏中,就会得到4根木棍.

    # 镐子

    这样做出镐子（什么工具都可以没有，镐子绝对要有）。可以把镐头换成石头或铁锭。

    然后去挖石头，这灰色的就是石头，用镐子采集，获得原石。

    这样做石镐，可以采集木镐采集不了的铁矿。

    有黑点点的是煤矿，可以挖出煤炭；

    带有暗金点点的是铁矿，可以挖出铁矿。

    # 火把

    这样做火把，用于照明。

    # 熔炉

    八个原石搭建一个熔炉，用来烧制熟食和矿物。

    # 铁锭

    煤炭、各种树叶以及大部分木制品可用作燃料，上方摆放烧制的物品，如图铁矿获得铁锭。

    初期没有煤炭，可以把树叶、木板和木质工具放进去当燃料烧，当然火一会就灭了，因为不耐烧，因此你需要多放一些燃料。

    # 铁镐

    如图这样做出铁镐，可采集铁矿以上更高级的矿物。

    # 斧头

    斧头的做法（石头可以换成木头，铁之类的，其他工具同理）主要用来砍木制品，也鼓励用作武器。

    # 剑

    剑的做法，可对多目标造成横砍的群伤效果。

    # 铲子

    铲子用来挖泥土，沙子，砂砾，速度更快。

    # 锄头

    锄头，种地用，对泥土右键可生成耕地。

    # 桶

    铁桶可以提水（在有水的地方按下左键挖水）。用右键（放置）倒水。

    # 耕地

    在耕地上播种（小麦种子可通过撸草获得）,耕地旁要有水源才能长好.
